# Paytm AI – Reputation Layer

**UPI Trust Score System** - Know who you're paying before you pay

A modern fintech application that analyzes UPI payment recipients and provides trust scores to help users make informed payment decisions.

## Features

- 🛡️ Real-time UPI trust analysis
- 📊 Comprehensive risk scoring
- ⚠️ Risk warnings for suspicious accounts
- 🎨 Modern, responsive UI with dark mode support
- 🔒 Secure payment verification

## Getting Started

### Prerequisites

- Node.js 18+
- npm or bun package manager

### Installation

```bash
npm install
# or
bun install
```

### Development

```bash
npm run dev
# or
bun run dev
```

### Build

```bash
npm run build
# or
bun run build
```

## Technology Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Radix UI Components
- Supabase (optional backend)

```
src/
```
